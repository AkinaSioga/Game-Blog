# 阿尔戈斯，行星核心攻略

阿尔戈斯，行星核心，来源于世界吞噬者（世吞）。

一言而蔽之：222分组，杀高等哥布林拿头颅，看球颜色去对应场地充能头颅，充完一起滋球，然后上平台打左右手后背头的红色弱点，然后输出。

提示：万神殿的BOSS会周期性的发射捕网，命中守护者会把人困猪然后往天上飞，记得救救你的好队友，不要让他out了。

## 场地结构

外圈由六块平台组成：

* 三个长弧形平台
* 三个小弧形平台

三个长平台是主要机制平台。每个长平台的中间偏外侧都有一个三角祭坛，也就是颅骨充能点。三角祭坛有三个角，每个角都可以用来充能颅骨。

三个小平台是高等哥布林刷新点。高等哥布林被击杀后会掉落颅骨，玩家需要把颅骨带到对应长平台的三角祭坛进行充能。

阿尔戈斯本体位于圆环中央，是一个巨大的 Vex 九头蛇。开场时，阿尔戈斯会被菱形屏障保护。每个长平台方向都能看到屏障的一面，每一面上方有三个三角节点。每轮机制中，只有其中一个面会刷出三个能量球。

## 推荐分工

推荐分工是：

* 三名守点
* 三名跑者

守点主要负责观察刷球面、清理小怪（会 刷 飞 龙）和报球体颜色。

跑者主要负责击杀高等哥布林、拾取颅骨、带到对应长平台充能，并在充能完成后参与射击能量球。

如果队伍熟练，也可以不固定死守点和跑者，但新手队伍建议先按 3 守点 + 3 跑者来打，沟通会更清楚。

## 破盾阶段流程

### 第一步：观察刷球面

开场后，先观察阿尔戈斯菱形屏障的三个面。

本轮只有一个面会刷出三个能量球。三个能量球一定出现在同一面，但颜色不一定都不同，也有可能重复。

例如可能出现：

* 紫 / 红 / 白
* 紫 / 紫 / 白
* 红 / 红 / 红

这里最重要的不是颜色本身，而是先确认「哪个面刷球」。确认刷球面之后，队伍就知道应该在哪个长平台方向集合破盾。

### 第二步：击杀高等哥布林，拿颅骨

三个小平台会刷新高等哥布林。

击杀高等哥布林后会掉落颅骨。跑者需要尽快拾取颅骨，并根据本轮刷出的能量球颜色，把颅骨带到对应的长平台三角祭坛充能。

例如：
（可以叫对应属性，也可以叫颜色 火/电/虚空 : 红/白/紫)
* 三个能量球是紫 / 橙 / 白：三个场地各充能一个
* 三个能量球是紫 / 紫 / 白：去紫场充能2火，白场充能1电
* 三个能量球是红 / 红 / 红：去红场地充能3个火

### 第三步：前往对应长平台充能

拿到颅骨后，跑者前往对应的长平台三角祭坛。

三角祭坛有三个角，每个角都可以充能。放置完后会自动充能，不用管它。

这一阶段需要注意两件事：

第一，颅骨充能完成后不要乱射，必须等三名玩家都准备好，再一起射击屏障上的能量球。

第二，如果没子弹了后续输出阶段也可以拿颅骨滋BOSS应急（原版本可用，万神殿不清楚是否还是高伤）

### 第四步：三人同时射击能量球

三名拿着充能颅骨的玩家站到能看到刷球面的方向。

每个人负责一个能球，用对应颜色的颅骨射击对应能量球。三颗球需要尽量同步击破。

如果三颗球都被正确击破，阿尔戈斯的菱形屏障会破裂，队伍进入下一阶段。

## 输出前处理阶段

屏障破裂后，圆环内部会刷出多层圆环平台。玩家需要通过这些平台移动到合适位置，攻击阿尔戈斯身上发红光的部位。

常见需要处理的部位包括：

* 左右手
* 头部
* 背部或肩部附近弱点

这一阶段的目标不是直接打血条，而是打掉阿尔戈斯的白盾。看到 Boss 身上有发红光的部位时，优先集火这些部位。白盾被破掉后，才进入真正的 Boss 输出阶段。

队伍可以分成左右两队：

* 左边三人负责左侧发光部位
* 右边三人负责右侧发光部位

如果本轮是头部或中间弱点，所有人可以集中火力处理。


## 输出阶段

白盾破掉后，阿尔戈斯进入可输出状态。

输出时建议所有人提前约定集合点，不要分散得太开。（基本上是左3右3，光井带汇编器，这样都可以吃到增伤。当然二光井也可以，但是汇编器都得带，毕竟目的还是为了能吃增伤。）

PS:这个阶段BOSS还是会发射捕网，记得救救你的好曼巴队友。
